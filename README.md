# basseProjekt
In this project we try to calculate odds of certain events happening in football matches based on statistics taken from official scorecards.


The app is built on a vue-typescript frontend with just a basic GUI showing the stats that will be calculated from the raw data. 
The backend is built using python, and to connect the two parts we use fastAPI.
Furthermore a postgreSQL database is used to store the information gathered from the scorecards. 

Original source statistics can be found on this website: https://www.football-data.co.uk/englandm.php
